To download data from ARC use the following command format in Windows Subsystem for Linux (WSL):

scp -r kbsanjayvasanth@datatransfer.arc.vt.edu: arc_path "local path"

Example:

scp -r kbsanjayvasanth@datatransfer.arc.vt.edu:/home/kbsanjayvasanth/Velocityraft2 "/mnt/e/no particle test smooth one case/"


Note: It will ask for a password, enter your login pwd and then you'd be prompted to approve the request in DUO mobile. Download your data to local hardrive after processing.
